Setup Instructions

Clone the repository, install dependencies with npm install, and run the development server with npm run dev from the project root to see the app locally. Ensure Node.js (v18+) is installed and configure a .env file with your MySQL credentials or adjust db.ts to match your database setup before running.

Features

Search books via the Open Library API, add items to wishlist/reading/completed lists, set reading goals, and track progress with simple CRUD operations and activity entries. Includes user authentication (register/login), profile and settings modals, and responsive UI components for both desktop and mobile.
